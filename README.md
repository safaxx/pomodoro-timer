# Pomodoro Timer

A single-screen Pomodoro timer web app built with React for the Dev Weekends assessment.

Live demo: https://pomodoro-timer-dw.netlify.app/

## Features

- Start, pause, resume, and reset timer
- Configurable focus and break durations
- Automatic focus-to-break and break-to-focus transitions
- Countdown display in `mm:ss`
- Audible cue when a session ends
- Focus completion message before break starts
- History of completed focus sessions for the current day
- History persists after page reload
- Responsive layout for desktop and mobile
